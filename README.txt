
Cada processo constitui de uma sequencia de caracteres de [1-255]
-----------------------------------------------------------------

São 3 processos

------------------------------------------------------------------

O último caracter deve ser a "#" e representa o termino do processo

-------------------------------------------------------------------

A entrada da sequencia é feita pelo treclado-constencia de dados

--------------------------------------------------------------------

O timeslice é digitado pelo usuário

---------------------------------------------------------------------

A execução utiliza uma fila circular

---------------------------------------------------------------------

A velocidade do programa é entrada pelo usuário [1-10]
onde cada unidade equivale a 1/10 seg.

----------------------------------------------------------------------

O programa deve apresentar na tela os 3 painéis de caracteres

-----------------------------------------------------------------------

O progama gera um txt com o log dos processos conforme exemplo do escalonador

----------------------------------------------------------------------------

Cada processo vai receber um parametro de repetição [1-100] digitado pelo usuário
